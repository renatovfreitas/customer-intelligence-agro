# Customer Intelligence for Agricultural Machinery Retail

This project analyzes sales data from an agricultural machinery retail business in Brazil.

The company sells agricultural machinery, tractor parts, garden equipment, oils, and related products. The goal of this project is to build a customer intelligence analysis focused on customer segmentation, sales behavior, profitability, and business opportunities.

## Project Goals

- Identify the most valuable customers
- Segment customers by revenue, frequency, recency, and profitability
- Analyze product groups and sales behavior
- Detect inactive or high-potential customers
- Generate business recommendations based on data

## Dataset

The initial dataset contains approximately 24,000 rows and 10 columns.

Each row represents one product sold inside a sale transaction.

## Columns

- Sale Item ID
- Sale Control Number
- Sale Date
- Customer Name
- Product ID
- Product Name
- Product Group
- Unit Sale Price
- Quantity
- Unit Cost

## Current Scope

This first version analyzes only product sales.

Services such as mechanics, lathe work, and welding are not included in the current scope.
